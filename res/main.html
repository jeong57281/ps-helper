split

C++

#include <iostream>
#include <sstream>

vector<string> split(string input, char delimiter){
    vector<string> ret;
    istringstream iss(input);
    string token;
    while(getline(iss, token, delimiter))
        ret.push_back(token);
    return ret;
}

분리하려는 문자열과 구분자를 전달받아 istringstream 을 이용해 문자열을 추출해내는 함수를 구현하여 사용한다.

C#
